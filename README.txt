# CSCG - Cardano Smart Contract Generator - WordPress Plugin

Welcome to the **CSCG - Cardano Smart Contract Generator** plugin for WordPress! This plugin allows seamless Cardano smart contract generation onto your WordPress site.

## Features

- **NFT Mint, Burn smart contract generation**
- **Vesting smart contract generation**
- **Utilities and assets files generation**
- **Smart contract onchain and ofchain generation**
- **Easy integration** with WordPress sites
- **Customization options** for seamless design fit

## Installation

1. Download the plugin zip file from [Github repository](https://github.com/EsubalewAmenu/CSCG-Cardano-Smart-Contract-Generator).
2. Log in to your WordPress admin panel.
3. Navigate to `Plugins > Add New`.
4. Click on the **Upload Plugin** button.
5. Choose the downloaded plugin zip file and click **Install Now**.
6. Once installed, click **Activate Plugin** to activate the plugin on your site.

## Configuration

1. Navigate to the pages and create one in your WordPress admin panel.
2. add [CSCG_form] shortcode.
3. Save your changes.
